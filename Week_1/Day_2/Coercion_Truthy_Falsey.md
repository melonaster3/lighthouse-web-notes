# Coercion and Truthy/Falsey

## Equal Signs 

=== means that it will compare both the values as well as the types. It will return false if 

``` javascript
23 === '23'
//This will be FALSE

23 == '23'
//This will be TRUE 
```

## Falsey Numbers

* 0 is a falsey Number
* "" is an empty string, falsey 
* null is empty value so falsey 
* undefined is falsey
* NaN not a number
